# SASS, Globbing, Auto prefixer, Concat, Minify, Uglyfy tools

###How to run

-Install node.js from here  [https://nodejs.org/download/](https://nodejs.org/download/)
-Use this commands 
```sh
$ gem install sass
$ npm install -g grunt-cli
``` 
-Run this command under the directory
```sh
$ npm install
``` 
-Grunt commands 

- grunt dev
- grunt watch
- grunt deploy
